# Meta X

## Getting Started
### 1. Clone this repo
```
$ git clone https://github.com/hidayatNgganteng/metaX.git MetaX
```
### 2. Install & Setup
```
$ cd MetaX

$ yarn install
```
### 3. Run Project
```
$ yarn run start
```
